# README

For all these the relevant subset of tasks (with more annotations and starter code) have been given please refer to it carefully if you choose to attempt, do not spend time fighting the PS.

In most cases you can find an appropriate `python3 main.py` or `python3 grade.py` to check your submission. Ensure you have a virtual environment containing (or global installation of) numpy, matplotlib and pandas (needed for data loading and plotting).

1. Lab0: Basic broadcasting, some template code is given so that you can directly run the code after filling the blanks. Read the PS for more info, and only focus on task 2.

2. Lab1: Basic LR/OLS solution. Quite approachable and you might learn a thing or two.

3. Lab4: Way beyond beginner or intermediate numpy; Some ML math stuff which you should def skip; can try out a few sections of the outlab part, it has been annotated for your ease!

4. Lab7: Want to implement FFNN with backprop from scratch? Mind altering stuff but fun.
